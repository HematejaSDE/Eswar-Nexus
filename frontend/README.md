# Eswar Nexus
### AI-Powered Campus Operating System for Eswar College of Engineering

Eswar Nexus is a modern full-stack campus management platform designed specifically for Eswar College of Engineering.  
The platform streamlines academics, placements, fee management, student services, and AI-powered assistance through one unified digital ecosystem.

---

# 🚀 Features

## 🔐 Authentication & Security
- JWT Authentication
- Role-Based Access Control
- Student / Faculty / Admin Login
- Protected Routes
- Password Encryption using bcrypt

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/eswar-nexus.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create `.env` inside backend folder:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

CLIENT_URL=http://localhost:5173
```
